# 🛒 Amazon Product Review Sentiment Classifier

This project implements a **Multi-Class Sentiment Analysis System** that classifies Amazon product reviews into:

- Positive 😊
- Neutral 😐
- Negative 😡

The system uses **Natural Language Processing (NLP)** and **Machine Learning** to analyze user input reviews and predict sentiment through an interactive **Streamlit web application**.

---

# 🚀 Features

- Multi-class sentiment classification (Positive / Neutral / Negative)
- Text preprocessing and cleaning
- TF-IDF feature extraction
- Machine learning model for sentiment prediction
- Interactive web interface using Streamlit
- Real-time sentiment prediction

---

# 🧠 Machine Learning Pipeline

1. Text preprocessing
2. Feature extraction using **TF-IDF**
3. Training machine learning classifier
4. Sentiment prediction
5. Web deployment using **Streamlit**

---

# 🛠 Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit
- TF-IDF Vectorization
